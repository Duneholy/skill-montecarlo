## Описание скилла

`montecarlo` — это навык (скилл) для агента, который помогает просчитать вероятность будущего. Он заставляет вас озвучить свои допущения, строит статистическую модель, локально прогоняет 100 000 итераций метода Монте-Карло и говорит вам суровую правду о шансах вашего плана на успех.

Он определяет скрытые переменные, которые с наибольшей вероятностью могут разрушить план, и предлагает одно конкретное действие с максимальным рычагом влияния.

## Как это работает

Скилл работает как строгая машина состояний:

1. **STATE A (Сбор данных)**: Агент задает вам до 6 вопросов простым языком, чтобы определить порог успеха, ваши лучшие/худшие оценки и известные риски.
2. **STATE B (Исполнение)**: Агент тайно добавляет собственные факторы риска (если вы упустили очевидные вещи) и генерирует `sim_config.json`. Затем он запускает Python-движок без внешних зависимостей (`run_sim.py`) на 100 000 итераций.
3. **STATE C (Отчет)**: "Билли Бин" выносит вердикт простым текстом: ваш % вероятности успеха, 95% доверительный интервал и конкретные факторы, которые привели к провалам в симуляции.

## Установка

Клонируйте этот репозиторий в папку со скиллами вашего агента:

```bash
# Codex
mkdir -p "${CODEX_HOME:-$HOME/.codex}/skills"
git clone https://github.com/Duneholy/montecarlo.git \
  "${CODEX_HOME:-$HOME/.codex}/skills/montecarlo"

# Claude Code
mkdir -p "$HOME/.claude/skills"
git clone https://github.com/Duneholy/montecarlo.git \
  "$HOME/.claude/skills/montecarlo"
```

## Использование

Вызовите скилл в интерфейсе вашего агента:

```text
/montecarlo
```

Или сразу опишите ситуацию:

```text
/montecarlo Хочу запустить новый продукт через 3 месяца. Бюджет 50k.
```

Агент перехватит управление и проведет вас через весь процесс симуляции.
